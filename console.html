Console function

console is used for log a information with help of diffent function supported by console tool

where we find?

u can find in chrome ->right click->inspect->console(ctrl+shift+j)

log() for displaying different value(like print)

error() default the error message will be highlighted with red color

warn() default the warning message will be highlighted with yellow color

clear() Console was cleared

time() and timeEnd() get time and process completed time(like start and stop)

table() showing in the table format

count() iteration count

group() and groupEnd() is separate block( time() and timeEnd())
